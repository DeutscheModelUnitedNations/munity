Informationen

Diese Seite dient der Bereitstellung von Software für den Einsatz auf Model United Nations Konferenzen. Die auf dieser Plattform angebotenen Dienste sind kostenfrei.

Die Serverbereitstellung erfolgt durch den Anbieter STRATO.

Auf dieser Website werden Cookies für die Erfassung von Sessions gespeichert.

Alle auf dieser Website eingegebenen und angesendeten Daten werden vom Server gespeichert und verarbeitet. Zudem werden Daten wie die IP-Addresse zwischengespeichert um WebSocket Dienste anzubieten und Schutzmaßnahmen gegen potentielle Angriffe einzuleiten.

Datenschutz

Der Administrator dieser Seite ist in der Lage alle eingegebenen Daten zu sehen.

Resolutionen, welche öffentlich freigeschaltet werden können auch von allen Benutzern dieser Website (obgleich diese ein Benutzerkonto haben) gesehen werden.

Die eingegebnen Daten werden im Rahmen dieser Anwendung gespeichert und für die weitere Verwendung aufbereitet. Eingegebnen Delegationen und Konferenzstrukturen können dazu verwendet werden Muster für die Erstellung anderer Kofnerenzen zu bieten.

Redelisten werden nach eine bestimmten Zeit wieder entfernt.

Die Plattform bietet einen Basisdatensatz zum Arbeiten mit Resolutionen und Redelisten für die Richtigkeit dieser Daten wird keine Haftung übernommen.
